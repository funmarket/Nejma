# **App Name**: Spotly

## Core Features:

- Video Feed: Display a continuous feed of videos from various artists.
- Category Tabs: Allow users to filter videos by category (Music, Acting, Creator, etc.).
- User Authentication: Enable users to connect their wallets.
- Profile Creation: Allow users to create and customize their profiles with bios, social links, and profile/banner photos.
- Video Submission: Enable artists to submit videos by providing YouTube links, descriptions, and categories.
- Voting and Bookmarking: Allow users to vote on videos and bookmark them.
- Generative AI assistant for Profile creation: Help users to create a bio, using a tool that enables to adjust various writing options. 

## Style Guidelines:

- Primary color: Deep violet (#6C47FF) for a vibrant, modern feel.
- Background color: Dark gray (#0A0A0A) for an immersive experience.
- Accent color: Electric blue (#14F195) to highlight interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif font known for its readability and modern, machined appearance.
- Use clean, minimalist icons from Lucide library for clear navigation.
- A dark, full-screen layout with clear sections for the video feed and profile information.
- Subtle transition animations on hover and interaction.
